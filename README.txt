1. Скачать Open Server, по ссылке (https://ospanel.io/) и установить его.
2. Перейти по пути Open Server -> OSPanel -> domains, перетащить папку froum с файлами по данному пути.
3. Перезапустить Open server.
4. Запустить Open server.
5. Перейти по ссылке http://127.0.0.1/openserver/phpmyadmin. Нажать вперед (логин: root, пароль: без пароля).
6. Нажать на кнопку "Создать БД", в поле имя базы данных ввести "froum", выбрать сравнение "utf8mb4_general_ci". Кнопка создать.
7. Перейти во вкладку Импорт. В пункте "Выберите файл" выбрать файл froum.sql (в папке сайта).


8. Открыть браузер, ввести в адресную строку froum/index.php
